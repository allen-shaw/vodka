# Vodka

a cmd tool based on [gin](https://gin-gonic.com/)

## Features

1. cmd line tools for building a web porject.
2. middleware

## Commands

### init

```bash
vodka init -idl=./idl/xxx.proto [-out=./]
```

### update

```bash
vodka update [-idl=./idl/xxx.proto]
```

### version

```bash
vodka version
vodka -v
```

### upgrade

```bash
vodka upgrade
```

### help

```bash
vodka help
vodka -h
```
